Master Chief Theme

Personal project

Master Chief is a Visual Studio (VS) Code Extension I created to celebrate the release of Halo Infinite. It has been inspired by the Master Chief's Mjolnir armour (and the pink and purple of the Needler).

To help put the theme together, I created an Adobe XD mockup of VS Code with different file types. This allows you to hot swap colours and build up colour palettes and styles quickly and easily using the eye dropper tool in XD. Using the repeatable components means you can update the parent component and the changes will cascade through to the child components.

The XD file can be found in the Github Repository.

Colours have been checked against the WebAIM contrast checker to try and ensure good contrast for usage. As much as possible I have achieved WCAG AAA contrast.
